This is a Realtime comment system using NodeJs, express, Mongo DB, Socket.io and vanila Javascript

This project aims to provide a platform for users to share their thoughts and ideas on various topics, and engage in real-time conversations with other users. With the help of Socket.io, users can view and respond to comments instantly without having to refresh the page. We have used MongoDB to store the comments and Express to build the server-side application. The front-end has been built using Vanilla JavaScript, which makes the application lightweight and efficient.
